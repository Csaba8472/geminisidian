# Weekly Synthesis Skill

Use this skill to analyze and synthesize a week's worth of notes and projects.

## Analysis Process

1. **Gather Week's Work**
   - All notes created/modified this week
   - Projects that saw activity

2. **Identify Patterns**
   - Recurring themes
   - Breakthrough moments
   - Energy patterns

3. **Synthesize Learning**
   - Key insights
   - How thinking evolved
   - Connections discovered

## Output Format

Create a weekly synthesis note with the following structure:

```markdown
# Weekly Synthesis - Week of [Date]

## Week at a Glance

- Notes created: [X]
- Projects active: [List]

## Key Themes

### Theme 1: [Name]

- Significance: [why it matters]
- Next actions: [what to do]

## Major Insights

1. [Insight with context]

## Next Week's Intentions

1. [Primary focus]
```
